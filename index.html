Launch settings

1. Re-launch unlocker as admin. Then check. (If nothing then go to next step.)
2. Re-launch unlocker as admin + Re-launch game as admin too.
3. DO NOT UNLOCK LUA ON GAME LOADSCREEN!! WAIT UNTILL GAME FULL LOAD!!!
4. If still not working. try to update your .Net

IN-Game Addon

1. Check where you put Feral addon.
2. Open Interface folder in game then open Addons.
3. Start game type in chat /fd help that will show all cmds.
4. If you want load profile type /fd list that will show profile list.
Now to load type: /fd load *here spec name*
(Example: If you see in list: Feral --- FlexHub.com , type /fd load Feral)
